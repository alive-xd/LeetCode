## Code Style

- Language: C++17
- STL used throughout (vectors, maps, sets, queues, etc.)
- Consistent formatting: 4-space indentation, descriptive variable names
- Each `solution.cpp` includes a comment block with:
  - Time complexity
  - Space complexity
  - Approach summary (1–2 lines)

### Example Header

```cpp
// Time:  O(n)
// Space: O(n)
// Approach: Hash map to store complement lookups in a single pass.

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int, int> map;
        for (int i = 0; i < nums.size(); i++) {
            int complement = target - nums[i];
            if (map.count(complement)) return {map[complement], i};
            map[nums[i]] = i;
        }
        return {};
    }
};
```
